常见问题

免费版有水印吗？

没有。所有方案导出的图片和 SVG 都不含水印，您可以自由使用。

可以随时升级或降级吗？

可以。升级立即生效，降级在当前计费周期结束后生效。您随时可以取消订阅。

云存储空间用来做什么？

用于保存您的标签云配置、设定、历史记录，以及跨设备同步数据。免费版 50MB 足够存储数千个标签云。

API 可以做什么？

API 允许您在自己的应用或网站中集成 3Dtag 的标签云生成功能。详细的 API 文档会在购买团队版后提供。

支持哪些支付方式？

我们支持支付宝、微信支付、信用卡（Visa/MasterCard/银联）。所有支付均通过加密通道处理。

有退款政策吗？

我们提供 14 天无条件退款保证。如果您对购买的专业版或团队版不满意，可在 14 天内申请全额退款。

准备好开始创作了吗？
