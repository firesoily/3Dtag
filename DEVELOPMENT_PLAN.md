  - 添加复合词保护列表（人工智能、机器学习等术语不被拆分）

### ✅ P1 - 交互增强 (20:00-21:30) ✅ COMPLETED

- [x] **用户设置持久化**
  - localStorage 存储主题、标签数量、上次输入、搜索关键词
  - 自动恢复：`_loadSettings()` 在 `init()` 调用
  - 自动保存：主题切换、生成成功后保存

- [x] **历史记录功能**
  - 保存最近 10 次生成（标签、原文、设置、统计）
  - `_saveToHistory()` / `_loadHistory()`

- [x] **搜索高亮与过滤优化**
  - 多关键词搜索（空格分隔，AND 逻辑）
  - 过滤时非匹配标签透明度降低至 0.2
  - 匹配计数实时显示 (`match-count / total-count`)
  - 主题切换保持过滤高亮状态

- [x] **动画效果增强**
  - 动画系统：`animatingTags` Map, `_updateAnimations()` 每帧更新
  - 入场动画：从 scale=0 展开，波浪效果（索引错开20ms，easeOutQuad）
  - 悬停放大：`_checkHover()` 检测，放大到 1.3 倍
  - 点击脉冲：`_triggerPulse()` 放大-缩小循环（300ms × 2）

### ✅ P2 - 导出增强 (21:00-22:00) ✅ COMPLETED

- [x] **SVG 矢量导出**
  - `exportSVGAsync()` 生成 SVG 字符串（可配置宽高、背景色）
  - 使用相同球面分布算法，XML 特殊字符转义
  - `exportSVG()` 方法（Blob 下载）

- [x] **PNG 高清导出**
  - 高清选项（2x 缩放）
  - 临时调整渲染尺寸，导出后恢复

- [x] **UI 更新**
  - 双按钮布局（PNG / SVG）
  - 高清 checkbox 选项

- [x] **自适应标签数量**
  - `_recommendTagLimit()` 智能算法
  - 基于中文/英文比例、字符数密度
  - 范围 20-100，自动推荐或用户手动覆盖
